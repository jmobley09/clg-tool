The Cable Length Generator is a robust Site Services tool for use in everyday operations. This tool comes with different features to automate and enhance the SS teams.


Click On Features to Find Out More!

The User inface will provide an easy and convienent way for users to operate the Tool. It will also allow for faster bug reporting, tracking, and resolution.

Length generation will be one of two major functions of the tool. It will take a patch plan, and return the lengthe of the cable needed for each line of the patch plan. This will allow the users to save by not having to manually measure each run.

Label generation will be the second major function of the tool. This take the patch plan provide and create all labels needed for each connection listed on the patch plan. This will be for all out of cab connections. Any in cab connections will need to be run seperately.

The final operation that the tool will complete is formatting the patch plan to standardized format. This will help all ADOC teams as they will be able to consistantly use the same patch plan. This will be instrumental in both operations and training.
